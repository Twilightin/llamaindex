サブエージェントは新しいターミナル不要です。今のセッションの中で使います。                                                
                                                                                                                            
  2つの使い方                                                                                                               
                                                                                                                            
  1. Claude が自動で起動する（推奨）                                                                                        
                                                                                                                            
  コードを書いた後、普通に話しかけるだけ：                                                                                  
                                                                                                                          
  「このコードをレビューして」                                                                                              
  「さっき書いたファイルを確認して」                                                                                        
                                                                                                                            
  code-reviewer の description に「コードを書いた後に自動起動」と設定されているので、Claude
  が適切なタイミングで自動的にサブエージェントを呼び出します。

  2. 明示的に指示する

  「code-reviewer エージェントでレビューして」
  「src/main.py を code-reviewer に見せて」

  ---
  流れのイメージ

  あなた → Claude Code（メインセッション）
                  ↓ 自動 or 指示
           code-reviewer サブエージェント起動
                  ↓
           コードを読んでレビュー結果を返す
                  ↓
           Claude Code がまとめてあなたに報告

  サブエージェントはバックグラウンドで動き、結果だけが返ってきます。別のウィンドウや操作は一切不要です。
